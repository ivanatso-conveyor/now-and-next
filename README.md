## What this is

A research-led redesign of the Conveyor Trust Center — the touchless surface every B2B security review starts with. Built from **9 interviews** and **135 sentiment ratings**, the work paired an interim ship with a Q2 vision sprint, holding two truths in tension: visitors wanted the fixes shipped now, sales wanted a swing at an AI-native rebuild.

The deck demonstrates strategic thinking, systems design, AI-native product craft, and measurable impact for product design leaders and recruiters.

## Tools & process

- **Research:** 9 customer interviews (Trust Center Admins, TPRM Visitors, Internal Sales/Marketing). FigJam synthesis. Sentiment ratings across 15 features × 9 voices = 135 data points.
- **Prototyping:** Two production-shaped prototypes (Card Layout vs Open Layout) built in **Lovable AI** in two days — a cost-of-testing collapse that let us test both directions instead of coin-flipping one.
- **Vision sprint:** A 3-day Q2 sprint with six designers, six prototypes, and one converged roadmap. My contribution was *Coco* — an AI agent + Trust Scorecard + comparison tools, prototyped in **Claude Code** and live at [ivanatso-conveyor.github.io/trust-center](https://ivanatso-conveyor.github.io/trust-center).

## Setup (enabling GitHub Pages)

This repo is designed to be served as a static site via GitHub Pages:

1. Go to **Settings → Pages** in this repository.
2. Under "Build and deployment," set the source to **Deploy from a branch**.
3. Pick the `main` branch and `/ (root)` folder.
4. Click **Save**.
5. Wait ~1 minute, then refresh. The live URL appears at the top of the Pages settings.
